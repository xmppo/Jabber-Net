GetOpt

Namespace: bedrock.util


GetOpt should be subclassed to create a class that handles command-line parameters. The subclass should use fields or properties that have the CommandLine attribute set on them. Fields and properties of type bool will be toggle flags, other types will take a value as either the next command-line parameter or following a colon.
