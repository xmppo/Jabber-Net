Jabber-Net

ItemList

Namespace: JabberNet.jabber.connection


Manages a list of items with a maximum size. Only one item with a given ID will be in the list at a given time.

Constructors

ConstructorDescription
new(node, maxItems)
Signature: (node:PubSubNode * maxItems:int) -> unit

Creates an item list, which will have at most some number of items.

Instance members

Instance memberDescription
Add(value)
Signature: value:obj -> int
Modifiers: abstract

Adds to the end of the list, replacing any item with the same ID, or bumping the oldest item if the list is full.

Returns: Index where the PubSubItem was inserted.
[()]
Signature: unit -> string

Gets or sets the contents of the specified item.

Returns: XmlElement representing the contents of the PubSubItem.
RemoveAt(index)
Signature: index:int -> unit
Modifiers: abstract

Makes sure that the underlying ID index is in sync when an item is removed.

RemoveId(id)
Signature: id:string -> unit

Removes the item with the given ID. No exception is thrown if no item is found with that ID.
